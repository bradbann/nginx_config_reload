# 获取配置
curl 127.0.0.1:8000/
```
{"ggpt": ["ip_hash", ["10.201.10.124:8080", "1", "10s"], ["10.201.10.224:8080", "1", "10s"]], "passport": ["ip_hash", ["10.201.10.126:8080", "1", "10s"], ["10.201.10.226:8080", "1", "10s"]]}
```
参数pretty可以生成可阅读格式

curl 127.0.0.1:8000/?pretty

# 查询指定 upstream 
curl 127.0.0.1:8000/pm?pretty
```
[
    [
        "10.201.10.125:8080",
        "1",
        "10s"
    ],
    [
        "10.201.10.225:8080",
        "1",
        "10s"
    ]
]

```
# 给指定 upstream 添加server
curl -XPOST 127.0.0.1:8000/pm?pretty -d '["10.201.10.233:3333", "111", "10s"]'


# 删除server
curl -XDELETE 127.0.0.1:8000/pm?pretty -d '10.201.10.233'
